# AI House Price Predictor 🏠
A Machine Learning web application that predicts house prices based on square footage and bedroom count using **Linear Regression**.

## 🚀 Features
- **Machine Learning:** Scikit-Learn Linear Regression model.
- **Backend:** Flask (Python) server to handle requests.
- **Frontend:** Modern, responsive UI with HTML5/CSS3 and FontAwesome icons.
- **Tools:** VS Code, Git, and Python.

## 🛠️ How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Train the model: `python model.py`
3. Start the server: `python app.py`
4. Open `http://127.0.0.1:5000` in your browser.
